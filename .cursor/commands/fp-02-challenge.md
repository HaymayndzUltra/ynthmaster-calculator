# FP-02: Challenge Operational Assumptions

Aggressively question every "Standard Procedure" to find **Asymmetric Advantages**.

## YOUR TASK

For each assumption in the synthesis/operation:

1. **Is this PHYSICALLY Mandatory?**
   - Does Thermodynamics require it? (Yes -> Keep)
   - Does the Law/Convention require it? (No -> Discard)

2. **What if we did the OPPOSITE?**
   - *Assumption:* "Buy Precursor A."
   - *Opposite:* "Synthesize Precursor A from scratch."
   - *Assumption:* "Cool with ice."
   - *Opposite:* "Run at higher temp with different catalyst."

3. **What if we REMOVED it?**
   - Remove the solvent? (Neat reaction?)
   - Remove the purification step? (Use dirty intermediate?)
