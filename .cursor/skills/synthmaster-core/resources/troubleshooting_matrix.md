# Tropical Synthesis Troubleshooting Matrix

| Issue | Observation | Likely Cause | Solution |
|-------|-------------|--------------|----------|
| Gummy Product | Crystals don't form; sticky mass | High humidity during gassing | Use a dehumidifier or air-con; double-dry solvents with MgSO4. |
| Rapid Temperature Spikes | Reaction mixture boils unexpectedly | High ambient temperature (35C+) | Use oversized ice-salt baths; add reagents in smaller increments. |
| Aluminum Passivation | Al squares don't react with HgCl2 | Protective oxide layer too thick | Wash Al with 10% NaOH until fizzing starts, then rinse and amalgamate. |
| Cloudy Extract | Organic layer is milky | Emulsion from excessive shaking | Add saturated Brine (NaCl solution) to break the emulsion. |
| Discolored Crystals | Yellow/Brown tint | Incomplete workup/impurities | Perform multiple washes with ice-cold anhydrous Acetone. |
| Low Conversion | Unreacted P2P detected | Weak Methylamine solution | Ensure Methylamine is freshly prepared or properly stored in a sealed container. |
